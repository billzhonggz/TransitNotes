# 运转笔记
一个允许交通迷计划、记录运转线路的网页APP。

[English](README.md)

## 意向功能列表

* 允许用户搜索多条公交/地铁等线路，并同时显示在地图中。
* 允许用户进行*设定途经点*的公交线路规划。
* 允许用户完全手动规划路线。
* 与配套的手机APP联动。

## 技术栈

* 高德地图JavaScript API
* React或Vue框架
